# 🚀 PrimeSpace Crash

PrimeSpace Crash adalah aplikasi web bertema cosmic yang terintegrasi dengan Telegram Bot.

## Features

- 🚀 Cosmic Space UI
- 👤 User Profile
- 📜 History System
- 🏆 Leaderboard
- 🤖 Telegram Bot
- 💰 Coin System
- 🛡️ Owner Panel
- 🚫 Ban System
- 📊 Statistics

## Commands

### User

```text
/start
/profile
/balance
/history
/play
```

### Owner

```text
/addcoin USER_ID AMOUNT
/removecoin USER_ID AMOUNT
/ban USER_ID
/unban USER_ID
/stats
```
